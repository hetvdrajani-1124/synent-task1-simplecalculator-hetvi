# Simple Calculator

## Objective
A simple Python calculator that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu

## Technologies Used
- Python

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:


py app.py



## Output

The program takes user input, performs the selected arithmetic operation, and displays the result.
